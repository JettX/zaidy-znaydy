# Зайди Знайди

Angular-сайт для магазину вінтажних побутових речей "Зайди Знайди" в Івано-Франківську.

## Розробка

```bash
npm install
npm start
```

Сайт буде доступний на `http://localhost:4200/`.

## Збірка

```bash
npm run build
```

Файли збірки будуть у `dist/zaidy-znaydy/`.

## Деплой на GitHub Pages

```bash
ng deploy --base-href=/zaidy-znaydy/
```

## Технології

- Angular 19
- SCSS
- TypeScript 5.7
